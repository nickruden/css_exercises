CSS Styles:
h2 { font-size: 20px; }
h2 { font-size: 30px; }

Заголовки h2 будут размером 30px, потому что заданы два селектора h2 с одинаковыми свойствами, и при таком раскладе, всегда будет приоритетнее тот селектор, который ниже.

Ex. 2

text

CSS Styles:
.h2 { font-size: 20px; color: blue; }
.h2 { font-style: italic; color: red; }

Заголовки h2 примут свойства обоих селекторов, не смотря на то, что они одинаковые, свойства у каждого из них разные, правило последнего селектора действует только тогда, когда селекторы одинаковые и их свойства также совпадают.
